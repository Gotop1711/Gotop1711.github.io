關於這份申請網頁，分享一個開發時的技術心得。
此網頁的外觀能夠切換 一般/深色 模式．一開始設想 CSS 做法的時候，由於是直接撰寫原始 CSS，沒有使用 SCSS 等預處理器，所以擔心要為了深色模式而對所有元素都再撰寫一次 class．還好上網研究之後，發現竟然忘記 CSS 的 variable 功能．製作深色模式其實只要再 :root 宣告好要切換模式的元素的色彩變數，在轉寫元素色彩相關屬性時都使用變數；另外，再撰寫一個 body 元素在深色模式時套用的 class，在此 class 裡改變 :root 設定好的色彩相關變數，最後再使用 js 去判斷點擊按鈕的事件去 添加/移除 body 的深色模式 class，就能製作出切換 一般/深色 模式的功能了．
